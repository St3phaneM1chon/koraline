BioCycle Peptides - Kit signature complète

Source utilisée:
- version horizontale complète du logo avec fond transparent

Contenu:
- master/: version source transparente recadrée
- web_headers/: formats pour site web, signature email, entêtes
- social_banners/: formats pour bannières de réseaux sociaux
- print/: fichiers HD 300 dpi pour impression

Recommandations:
- Signature email: signature_600x200.png ou signature_1200x400.png
- Site web / entête: signature_1600x500.png ou signature_2400x800.png
- LinkedIn / X / Facebook / YouTube: utiliser les fichiers du dossier social_banners
- Impression: utiliser les fichiers du dossier print
- Imprimeur: signature_print_5000x1600_300dpi.tif

Note:
- Fond transparent sur les PNG et TIFF
- Une version JPG fond blanc est aussi fournie pour les usages bureautiques
- Pour du très grand format premium, un vrai vectoriel SVG/AI/EPS reste préférable
