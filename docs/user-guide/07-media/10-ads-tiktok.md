# Publicites TikTok

> **Section**: Media > Publicites > TikTok Ads
> **URL**: `/admin/media/ads-tiktok`
> **Niveau**: Intermediaire
> **Temps de lecture**: ~12 minutes

---

## A quoi sert cette page ?

La page **TikTok Ads** affiche le tableau de bord de vos campagnes publicitaires TikTok. Elle centralise les metriques de performance de vos annonces video courtes.

**En tant que gestionnaire, vous pouvez :**
- Voir les metriques globales : impressions, clics, depenses, conversions, CTR et CPA
- Suivre l'evolution quotidienne des performances
- Consulter la liste de vos campagnes actives
- Synchroniser les donnees depuis TikTok Ads Manager
- Acceder au portail TikTok Ads pour gerer vos campagnes

---

## Concepts cles pour les debutants

| Terme | Definition |
|-------|-----------|
| **In-Feed Ad** | Annonce video qui apparait dans le flux "Pour toi" des utilisateurs |
| **Spark Ad** | Format publicitaire utilisant un contenu organique existant |
| **TikTok Ads Manager** | Portail de gestion des campagnes publicitaires TikTok |

---

## Comment y acceder

1. Allez dans **Media > Tableau de Bord**
2. Cliquez sur la carte **TikTok** dans la section "APIs & Publicites"
3. Ou naviguez vers `/admin/media/ads-tiktok`

---

## Vue d'ensemble de l'interface

L'interface suit le modele standard des pages publicitaires :

### Cartes KPI (6 indicateurs)
Impressions, Clics, Depenses, Conversions, CTR, CPA.

### Graphique de tendance quotidienne
Evolution jour par jour avec possibilite de changer la periode.

### Liste des campagnes
Tableau detaille par campagne.

---

## Fonctionnalites detaillees

### Synchroniser les donnees
1. Cliquez sur **Synchroniser**
2. Les donnees sont recuperees via l'API TikTok Ads
3. La derniere synchronisation est mise a jour

### Benchmarks TikTok Ads
- **CTR moyen** : 0.5% a 1.5% (varie selon le secteur)
- **Cout par clic moyen** : 0.50$ a 2.00$ CAD
- **Meilleur format** : videos courtes (15-30 secondes) avec accroche dans les 3 premieres secondes

---

## Cas d'utilisation pour BioCycle Peptides

- **Video educative** : expliquer les peptides de recherche en 30 secondes
- **Temoignages** : clients partageant leur experience de recherche
- **Promotions flash** : annoncer une offre speciale avec un code promo

---

## Questions frequentes

**Q : TikTok est-il pertinent pour les peptides de recherche ?**
R : TikTok attire une audience plus jeune et scientifiquement curieuse. Les videos educatives courtes sur les peptides performent bien sur cette plateforme.

**Q : Comment creer une campagne TikTok Ads ?**
R : Les campagnes se creent dans TikTok Ads Manager (ads.tiktok.com). Koraline affiche les resultats.
