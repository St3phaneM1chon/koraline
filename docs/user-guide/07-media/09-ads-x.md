# Publicites X (Twitter)

> **Section**: Media > Publicites > X Ads
> **URL**: `/admin/media/ads-x`
> **Niveau**: Intermediaire
> **Temps de lecture**: ~12 minutes

---

## A quoi sert cette page ?

La page **X (Twitter) Ads** affiche le tableau de bord de vos campagnes publicitaires sur X (anciennement Twitter). Elle centralise les metriques de performance de vos annonces.

**En tant que gestionnaire, vous pouvez :**
- Voir les metriques globales : impressions, clics, depenses, conversions, CTR et CPA
- Suivre l'evolution quotidienne des performances
- Consulter la liste de vos campagnes actives
- Synchroniser les donnees depuis X Ads
- Acceder au portail X Ads pour gerer vos campagnes

---

## Concepts cles pour les debutants

| Terme | Definition |
|-------|-----------|
| **Promoted Tweet** | Publication sponsorisee qui apparait dans le fil d'actualite des utilisateurs cibles |
| **Engagement** | Interactions avec vos annonces (retweets, likes, clics, reponses) |
| **Impressions** | Nombre de fois que votre annonce a ete affichee dans un fil |

---

## Comment y acceder

1. Allez dans **Media > Tableau de Bord**
2. Cliquez sur la carte **X (Twitter)** dans la section "APIs & Publicites"
3. Ou naviguez vers `/admin/media/ads-x`

---

## Vue d'ensemble de l'interface

L'interface suit le meme modele que les autres pages de publicites :

### 1. En-tete avec controles
- Nom de la plateforme et icone
- Derniere synchronisation
- Bouton **Synchroniser** et selecteur de periode (7/30/90/365 jours)

### 2. Cartes KPI (6 indicateurs)

| KPI | Description |
|-----|-------------|
| **Impressions** | Affichages de vos annonces |
| **Clics** | Clics sur vos annonces |
| **Depenses** | Budget consomme (CAD) |
| **Conversions** | Actions completees |
| **CTR** | Taux de clics |
| **CPA** | Cout par acquisition |

### 3. Graphique de tendance quotidienne
Barres montrant l'evolution jour par jour.

### 4. Liste des campagnes
Tableau avec metriques individuelles par campagne.

---

## Fonctionnalites detaillees

### Synchroniser les donnees
1. Cliquez sur **Synchroniser** en haut a droite
2. Les donnees sont recuperees via l'API X Ads
3. Message de confirmation a la fin

### Benchmarks pour X Ads
- **CTR moyen** pour l'e-commerce : 0.8% a 1.5%
- **CPA acceptable** : varie selon la valeur moyenne de votre panier
- **Meilleur jour de publication** : mardi et mercredi

---

## Flux de travail recommandes

### Suivi hebdomadaire
1. Selectionnez **7 jours**
2. Comparez le CTR avec la semaine precedente
3. Identifiez les campagnes a ajuster dans le portail X Ads
