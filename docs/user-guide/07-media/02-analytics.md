# Analytique Media

> **Section**: Media > Analytique
> **URL**: `/admin/media/analytics`
> **Niveau**: Intermediaire a avance
> **Temps de lecture**: ~15 minutes

---

## A quoi sert cette page ?

La page **Analytique Media** fournit des graphiques et metriques detailles sur les performances de vos contenus mediatiques : videos, publications sur les reseaux sociaux, et campagnes publicitaires.

**En tant que gestionnaire, vous pouvez :**
- Voir le nombre total de vues, clics, partages et conversions
- Suivre l'evolution quotidienne de l'engagement sur une periode configurable (7, 30, 90 ou 365 jours)
- Analyser la repartition des publications par plateforme (Facebook, Instagram, X, TikTok, LinkedIn)
- Identifier vos contenus les plus performants
- Mesurer votre taux d'engagement global

---

## Concepts cles pour les debutants

| Terme | Definition |
|-------|-----------|
| **Impressions** | Nombre de fois qu'un contenu a ete affiche (meme sans clic) |
| **Taux d'engagement** | Pourcentage de personnes qui interagissent avec votre contenu (clics + partages / vues) |
| **Conversion** | Action mesurable generee par un contenu (achat, inscription, telechargement) |
| **CTR** | Click-Through Rate, taux de clics par rapport aux impressions |

---

## Comment y acceder

1. Connectez-vous a l'administration (`/admin`)
2. Naviguez vers **Media** dans la barre de navigation
3. Cliquez sur **Analytique** dans le panneau lateral

---

## Vue d'ensemble de l'interface

### 1. En-tete et selecteur de periode
- Le titre **Analytique** avec le taux d'engagement global affiche en gras
- Un menu deroulant a droite pour choisir la periode : **7 jours**, **30 jours**, **90 jours** ou **365 jours**

### 2. Cartes KPI (4 indicateurs)

| KPI | Description | Icone |
|-----|-------------|-------|
| **Vues totales** | Nombre cumule de vues sur la periode | Oeil |
| **Clics totaux** | Nombre de clics generes | Pointeur |
| **Partages totaux** | Nombre de fois que le contenu a ete partage | Partage |
| **Conversions** | Actions concretes generees par le contenu | Tendance |

### 3. Graphique de tendance quotidienne (Area Chart)
Un graphique en aires superposees montrant l'evolution jour par jour de :
- **Vues** (bleu)
- **Clics** (vert)
- **Partages** (violet)

### 4. Repartition par plateforme (Pie Chart)
Un diagramme circulaire montrant la proportion de publications par plateforme :
- Facebook (bleu)
- Instagram (rose)
- X/Twitter (bleu clair)
- TikTok (noir)
- LinkedIn (bleu fonce)

Sous le graphique, un resume indique pour chaque plateforme le ratio publie/total.

### 5. Contenu le plus performant (Bar Chart + tableau)
- Un graphique en barres comparant les 10 meilleurs contenus sur les metriques vues, clics, partages et conversions
- Un tableau detaille avec le classement, le lien vers la video, le type de contenu et le taux d'engagement

---

## Fonctionnalites detaillees

### Changer la periode d'analyse
1. Cliquez sur le menu deroulant en haut a droite
2. Selectionnez la periode souhaitee (7, 30, 90 ou 365 jours)
3. Tous les graphiques et KPI se rechargent automatiquement

### Analyser un contenu specifique
1. Dans le tableau "Top Content", reparez le contenu qui vous interesse
2. Cliquez sur le lien du contenu (identifiant cliquable en bleu)
3. Vous etes redirige vers la page de detail de la video

---

## Flux de travail recommandes

### Rapport hebdomadaire
1. Selectionnez la periode **7 jours**
2. Notez les KPI principaux (vues, clics, conversions)
3. Identifiez le contenu le plus performant
4. Comparez avec la semaine precedente (selectionnez 30 jours pour voir la tendance)

### Analyse mensuelle
1. Selectionnez **30 jours**
2. Observez la repartition par plateforme : investissez-vous suffisamment sur chaque canal ?
3. Analysez le taux d'engagement : un taux superieur a 3% est considere comme bon
4. Identifiez les contenus a faible engagement pour les ameliorer ou les retirer

---

## Questions frequentes

**Q : Les donnees sont vides, il n'y a rien a afficher**
R : Les analytiques se remplissent au fur et a mesure que vous publiez du contenu via le planificateur social ou que vos videos accumulent des vues. Commencez par publier du contenu.

**Q : Le taux d'engagement semble faible**
R : Un taux entre 1% et 3% est normal pour la plupart des industries. Au-dessus de 3%, c'est excellent. En dessous de 1%, revoyez votre strategie de contenu.

## Glossaire

| Terme | Definition |
|-------|-----------|
| **KPI** | Key Performance Indicator, indicateur cle de performance |
| **Area Chart** | Graphique en aires, montrant l'evolution dans le temps |
