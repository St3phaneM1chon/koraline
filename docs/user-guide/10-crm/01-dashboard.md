# Tableau de Bord CRM

> **Section**: CRM > Tableau de bord
> **URL**: `/admin/crm`
> **Niveau**: Debutant a avance
> **Temps de lecture**: ~15 minutes

---

## A quoi sert cette page ?

Le **Tableau de bord CRM** est votre centre de commande pour toute l'activite commerciale de BioCycle Peptides. Il offre une vue synthetique de la relation client, des opportunites de vente et de la performance de votre equipe.

**En tant que gestionnaire, vous pouvez :**
- Voir les indicateurs cles de performance (KPIs) en un coup d'oeil
- Suivre le nombre de contacts, leads et deals actifs
- Consulter le pipeline des ventes en temps reel
- Identifier les opportunites qui progressent et celles qui stagnent
- Visualiser les revenus generes par periode (jour, semaine, mois, trimestre)
- Suivre l'activite de votre equipe (appels, emails, taches completees)
- Acceder rapidement aux actions prioritaires (leads chauds, taches en retard)
- Comparer les performances entre periodes

## Concepts de base pour les debutants

### Qu'est-ce qu'un CRM ?

Un CRM (Customer Relationship Management, ou Gestion de la Relation Client) est un outil qui centralise toutes les interactions entre votre entreprise et ses clients ou prospects. Imaginez un carnet d'adresses ultra-intelligent qui se souvient de chaque conversation, chaque achat, chaque email echange.

### Pourquoi c'est important ?

- **Rien ne se perd** : chaque interaction est enregistree
- **Suivi structure** : vous savez exactement ou en est chaque prospect
- **Meilleure conversion** : en suivant un processus, vous vendez plus
- **Vision globale** : la direction voit la sante commerciale en temps reel

### Vocabulaire essentiel

| Terme | Definition |
|-------|-----------|
| **Contact** | Une personne physique dans votre base (client ou prospect) |
| **Lead** | Un contact qui a montre de l'interet mais n'a pas encore achete |
| **Deal** | Une opportunite de vente en cours de negociation |
| **Pipeline** | L'ensemble des etapes par lesquelles passe un deal, de la decouverte a la conclusion |
| **Conversion** | Le passage d'un prospect a un client (il a achete) |
| **Churn** | Un client qui cesse d'acheter ou se desabonne |
| **CLV** | Customer Lifetime Value : valeur totale qu'un client genere dans sa vie |

## Comment y acceder

### Methode 1 : Via le menu principal
1. Connectez-vous a l'interface d'administration (`/admin`)
2. Dans la **barre de navigation horizontale** en haut, cliquez sur **CRM**
3. Le tableau de bord s'affiche automatiquement (c'est la page par defaut de la section)

### Methode 2 : Via le rail de navigation
1. Dans la colonne d'icones a gauche, cliquez sur l'icone **CRM** (entonnoir ou cible)
2. Le panneau CRM s'ouvre, le tableau de bord est la premiere option

### Methode 3 : Via la barre de recherche
1. Cliquez sur la barre de recherche en haut (ou tapez `/`)
2. Tapez "CRM" ou "tableau de bord CRM"
3. Selectionnez le resultat

## Vue d'ensemble de l'interface

L'interface du tableau de bord CRM est divisee en **4 zones principales** :

### 1. Les cartes de KPIs (en haut)

Cinq cartes resumant l'etat de votre CRM :

| Carte | Description |
|-------|-------------|
| **Contacts totaux** | Nombre total de contacts dans votre base |
| **Leads actifs** | Nombre de leads en cours de qualification |
| **Deals ouverts** | Nombre d'opportunites en cours dans le pipeline |
| **Revenu pipeline** | Valeur totale des deals ouverts (en $CA) |
| **Taux de conversion** | Pourcentage de leads convertis en clients |

Chaque carte affiche egalement la **variation** par rapport a la periode precedente (fleche verte pour une hausse, rouge pour une baisse).

### 2. Le graphique de pipeline (au centre)

Un graphique en entonnoir ou en barres montrant la repartition des deals par etape :
- **Prospection** : premiers contacts, decouverte
- **Qualification** : besoin identifie, budget confirme
- **Proposition** : offre envoyee
- **Negociation** : discussion sur les termes
- **Conclusion** : deal gagne ou perdu

### 3. Les activites recentes (colonne droite)

Un fil d'actualite montrant les dernieres actions :
- Nouveaux leads entrants
- Deals qui ont change d'etape
- Taches completees par l'equipe
- Emails envoyes et recus
- Appels effectues

### 4. Les listes prioritaires (en bas)

- **Leads chauds** : les prospects avec le meilleur score, a contacter en priorite
- **Taches en retard** : les actions qui auraient du etre completees
- **Deals proches de la cloture** : opportunites a finaliser rapidement
- **Relances a faire** : contacts qui n'ont pas ete sollicites depuis un moment

## Fonctionnalites detaillees

### 1. Filtrer par periode

**Objectif** : Comparer les performances sur differentes periodes.

**Etapes** :
1. En haut a droite du tableau de bord, trouvez le selecteur de periode
2. Choisissez parmi : Aujourd'hui, Cette semaine, Ce mois, Ce trimestre, Cette annee, Personnalise
3. Les KPIs et graphiques se mettent a jour automatiquement

> **Astuce** : Le mode "Personnalise" permet de definir une plage de dates precise, utile pour comparer des campagnes specifiques.

### 2. Naviguer rapidement vers une section

**Objectif** : Acceder directement a une sous-section du CRM depuis le tableau de bord.

**Etapes** :
1. Cliquez sur le titre de n'importe quelle carte de KPI pour acceder a la page correspondante
2. Par exemple, cliquer sur "Leads actifs" vous emmene directement a la page Leads
3. Cliquer sur "Deals ouverts" ouvre la page Pipeline

### 3. Personnaliser les widgets

**Objectif** : Adapter le tableau de bord a vos besoins.

**Etapes** :
1. Cliquez sur le bouton **Personnaliser** (icone engrenage) en haut a droite
2. Glissez-deposez les widgets pour reorganiser l'affichage
3. Activez ou desactivez les widgets selon vos preferences
4. Cliquez sur **Sauvegarder la disposition**

## Workflows courants

### Revue matinale des ventes
1. Ouvrez le tableau de bord CRM
2. Verifiez les **leads chauds** : y a-t-il de nouveaux prospects a contacter ?
3. Consultez les **taches en retard** : quelles actions sont prioritaires ?
4. Regardez les **deals proches de la cloture** : quelles ventes peuvent etre finalisees ?
5. Parcourez les **activites recentes** : que s'est-il passe hier ?

### Revue hebdomadaire de performance
1. Selectionnez la periode "Cette semaine"
2. Comparez le taux de conversion avec la semaine precedente
3. Verifiez la valeur du pipeline : est-elle en croissance ?
4. Identifiez les deals bloques depuis plus de 7 jours
5. Planifiez les actions de la semaine suivante

## Questions frequentes (FAQ)

**Q : Les donnees du tableau de bord sont-elles en temps reel ?**
R : Oui, les KPIs se mettent a jour en temps reel. Les graphiques et rapports se rafraichissent automatiquement toutes les quelques minutes.

**Q : Puis-je exporter les donnees du tableau de bord ?**
R : Oui, utilisez le bouton **Exporter** en haut a droite pour telecharger un rapport PDF ou CSV des KPIs affiches.

**Q : Le tableau de bord est-il le meme pour tous les utilisateurs ?**
R : Par defaut oui, mais chaque utilisateur peut personnaliser sa disposition. Les administrateurs peuvent aussi creer des tableaux de bord partages via le Dashboard Builder (voir page dediee).

**Q : Que signifie un taux de conversion en baisse ?**
R : Cela peut indiquer plusieurs choses : leads de moindre qualite, processus de vente trop long, manque de suivi. Utilisez les rapports detailles pour diagnostiquer la cause.

## Glossaire

| Terme | Definition |
|-------|-----------|
| **KPI** | Key Performance Indicator, indicateur cle de performance |
| **Pipeline** | Representation visuelle du parcours d'un deal de la prospection a la conclusion |
| **Lead scoring** | Note automatique attribuee a un prospect selon ses actions et son profil |
| **Taux de conversion** | Pourcentage de prospects qui deviennent clients |
| **Revenu pipeline** | Somme des montants de tous les deals ouverts |
| **Churn rate** | Taux d'attrition, pourcentage de clients qui partent |
