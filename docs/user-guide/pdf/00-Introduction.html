Introduction a la Suite Koraline

Temps de lecture: ~10 minutes

Bienvenue dans Koraline

Koraline est la suite d'administration complete de BioCycle Peptides. C'est votre outil central pour gerer l'ensemble de votre entreprise : ventes, produits, clients, marketing, emails, telephonie, comptabilite, et bien plus.

Ce guide utilisateur vous accompagne pas a pas dans chaque fonctionnalite. Que vous soyez debutant ou avance, chaque page est documentee avec des explications claires, des captures d'ecran et des scenarios concrets.

L'interface Koraline — Style Outlook

L'interface de Koraline est inspiree de Microsoft Outlook. Voici les zones principales :

1. Barre de navigation horizontale (en haut)

Les 12 sections sont accessibles via des boutons-onglets en haut de l'ecran. Cliquez sur une section pour l'ouvrir.

2. Barre de ruban (Ribbon)

Juste en dessous, la barre de ruban contient les actions contextuelles de la page en cours. Les boutons changent selon la page. Par exemple, sur la page Commandes, vous aurez "Nouvelle commande", "Supprimer", "Exporter".

3. Rail de navigation (colonne d'icones a gauche)

Une colonne etroite avec des icones pour chaque section. Cliquez sur une icone pour ouvrir la section correspondante.

4. Panneau de section (sidebar gauche)

Quand une section est ouverte, un panneau lateral affiche la liste des sous-pages. Par exemple, la section Commerce affiche : Commandes, Clients, Distributeurs, Abonnements, Inventaire, Fournisseurs.

5. Zone principale (maitre/detail)

La majorite des pages utilisent une disposition en deux panneaux :

  • Panneau gauche (maitre) : Liste des elements (commandes, produits, contacts...)
  • Panneau droit (detail) : Detail complet de l'element selectionne

6. Barre d'outils superieure

En haut a droite, des boutons utilitaires :

  • Copilote IA : Assistant intelligent (Aurelia)
  • Densite d'affichage : Compact ou confortable
  • Mode sombre/clair : Changer le theme visuel
  • Notifications : Alertes en temps reel
  • Commandes rapides : Acces direct aux commandes
  • Chat support : Acces direct au chat

Comment utiliser ce guide

Lire dans l'ordre

Si vous decouvrez Koraline, nous recommandons de commencer par :

  1. Tableau de bord — pour comprendre l'interface
  2. Commerce > Commandes — la page la plus utilisee
  3. Catalogue > Produits — gerer vos produits
  4. Puis explorez les autres sections selon vos besoins

Lire par besoin

Chaque page est independante. Si vous cherchez comment faire une action specifique, allez directement a la page concernee.

Structure de chaque guide

Chaque page de ce guide suit le meme format :

  1. A quoi sert cette page — Resume rapide
  2. Concepts cles — Explications pour debutants
  3. Comment y acceder — 2-3 methodes de navigation
  4. Vue d'ensemble — Capture d'ecran + description des zones
  5. Fonctions detaillees — Chaque bouton et action explique
  6. Workflows — Scenarios business complets
  7. FAQ — Questions frequentes
  8. Glossaire — Definitions des termes techniques
  9. Pages liees — Liens vers les pages associees

Se connecter a Koraline

  1. Ouvrez votre navigateur web
  2. Allez a l'adresse de votre boutique suivie de /admin (ex: biocyclepeptides.com/admin)
  3. Connectez-vous avec votre email et mot de passe
  4. Ou utilisez l'un des boutons de connexion sociale (Google, Microsoft, Apple, X)

Astuce : Mettez la page /admin en favoris dans votre navigateur pour y acceder rapidement.

Obtenir de l'aide

  • Dans l'application : Cliquez sur le bouton Copilote IA en haut a droite pour poser une question a Aurelia
  • Ce guide : Consultez la page correspondante a votre question
  • Support : Utilisez le chat support ou envoyez un email a l'equipe technique
