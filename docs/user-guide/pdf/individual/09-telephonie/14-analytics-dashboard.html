Tableau de bord Analytics

Section: Telephonie > Analytics

Concepts pour debutants

Le tableau de bord Analytics est le point d'entree vers les quatre sous-sections

d'analyse de la telephonie. Il offre une vue d'ensemble avec les metriques principales

et des liens directs vers les analyses detaillees.

Ce hub permet de decider rapidement quelle analyse approfondir : performance des agents,

des files d'attente, des appels en general, ou l'analyse vocale par IA.

Comment y acceder

  1. Dans le panneau d'administration, cliquez sur l'icone Telephone dans la barre laterale.
  2. Sous Avance, cliquez sur Analytics.
  3. L'URL directe est : /admin/telephonie/analytics

Apercu de l'interface

Cartes de navigation

Quatre cartes, chacune representant une sous-section :

CarteDescriptionMetrique principale

Chaque carte affiche :

  • Titre et description
  • Icone representative
  • Metrique principale avec libelle
  • Lien "Voir" vers la page detaillee

Metriques de resume

Un score de satisfaction moyen global est calcule a partir des sondages post-appel.

Fonctions detaillees

Navigation rapide

Cliquez sur n'importe quelle carte pour acceder directement a la section d'analyse

correspondante. Les donnees affichees sur les cartes sont un resume de la base complete.

Workflows courants

Workflow 1 : Revue hebdomadaire de performance

  1. Ouvrez le tableau de bord Analytics.
  2. Consultez les metriques de resume pour identifier les tendances.
  3. Cliquez sur Performance des agents pour les details individuels.
  4. Cliquez sur Analytics files pour evaluer les temps d'attente.

FAQ

Q : Les donnees sont-elles en temps reel ?

R : Les metriques affichees sont calculees a partir de la base de donnees et refletent

les dernieres donnees disponibles au moment du chargement.
