Brand Kit

Section: Media > Gestion de Contenu > Brand Kit

URL: /admin/media/brand-kit

Niveau: Debutant a intermediaire

Temps de lecture: ~15 minutes

A quoi sert cette page ?

Le Brand Kit (Kit de Marque) centralise l'identite visuelle de BioCycle Peptides : couleurs, typographies, logo et directives de marque. C'est la reference unique pour tous les supports de communication.

En tant que gestionnaire, vous pouvez :

  • Consulter la palette de couleurs complete avec codes hexadecimaux
  • Copier une couleur en un clic pour l'utiliser dans vos designs
  • Voir les typographies de marque avec exemples
  • Consulter les versions du logo (fond clair et fond sombre)
  • Lire les directives d'utilisation du logo
  • Modifier les couleurs, polices et directives (mode edition)
  • Sauvegarder les modifications

Comment y acceder

  1. Allez dans Media dans la barre de navigation
  2. Cliquez sur Brand Kit dans le panneau lateral
  3. Ou naviguez vers /admin/media/brand-kit

Fonctionnalites detaillees

Copier une couleur

  1. Survolez un carre de couleur
  2. L'icone de copie apparait
  3. Cliquez pour copier le code hexadecimal
  4. Un message confirme la copie avec le nom de la couleur

Modifier le Brand Kit

  1. Cliquez sur Modifier en haut a droite
  2. Le panneau d'edition s'ouvre
  3. Modifiez le nom de marque, les couleurs ou les directives
  4. Cliquez sur Sauvegarder
  5. Les modifications sont appliquees immediatement

Annuler les modifications

  1. En mode edition, cliquez sur Annuler
  2. Les valeurs reviennent a leur etat precedent

Flux de travail recommandes

Mise en place initiale

  1. Definissez votre couleur primaire (vert emeraude BioCycle par defaut)
  2. Definissez votre couleur d'accent (violet par defaut)
  3. Redigez vos directives de marque (utilisation du logo, interdictions)
  4. Sauvegardez

Coherence de marque

  • Avant de creer un visuel marketing, consultez le Brand Kit
  • Copiez les couleurs exactes pour vos publications sur les reseaux sociaux
  • Utilisez les polices specifiees dans vos documents
  • Respectez les versions du logo (fond clair/sombre)

Questions frequentes

Q : Les couleurs du Brand Kit s'appliquent-elles automatiquement au site ?

R : Les couleurs primaire et d'accent sont utilisees dans le theme du site. Les modifications de couleur dans le Brand Kit sont refletes progressivement dans l'interface.

Q : Puis-je ajouter d'autres polices ?

R : Les polices affichees ici sont celles configurees dans le theme. Pour changer de police, contactez votre administrateur technique.

Q : Comment telecharger le logo en haute resolution ?

R : Les logos affiches ici sont des apercu textuels. Pour les fichiers logo (PNG, SVG), consultez la Mediatheque et cherchez dans le dossier "general" ou "images".
